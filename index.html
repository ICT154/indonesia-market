API to get Indonesia stock market and mutual fund data.

Stock Market

1. List of Stock

https://indonesia-market.vercel.app/api/stocks

Query Parameter

Parameter Description
search Type: string
Search query
page Type: int | Default: 1
Page to retrieve
per_page Type: int | Default: 25
Number of records per page
sort_by Type: string | Default: Code
Sort the results by the given field. Possible values: Code, Name, SectorName, SubSectorName, Last, AdjustedClosingPrice, PrevClosingPrice, AdjustedOpenPrice, AdjustedHighPrice, AdjustedLowPrice, Per, Pbr, Volume, Value, Frequency, OneDay, OneWeek, OneMonth, ThreeMonth, SixMonth, OneYear, ThreeYear, FiveYear, TenYear, Mtd, Ytd, Capitalization, BetaOneYear, StdevOneYear, LastDate,
sort_direction Type: string | Default: asc
Sort direction: asc or desc.
index_id Type: int
Filter stocks by the given stock index id. Get all possible stock index ids from the List of Stock Index.
sector_id Type: int
Filter stocks by the given sector id. Get all possible sector ids from the List of Sector.
subsector_id Type: int
Filter stocks by the given subsector id. Get all possible subsector ids from the List of Sector.

Mutual Funds

1. List of Mutual Fund

https://indonesia-market.vercel.app/api/mutual-funds

Query Parameter

Parameter Description
search Type: string
Search query
page Type: int | Default: 1
Page to retrieve
per_page Type: int | Default: 25
Number of records per page
sort_by Type: string | Default: Code
Sort the results by the given field.
Possible values: Name, InvestmentManagerName, CustodianBankName, Type, Currency, Dividend, NetAssetValue, DailyReturn, WeeklyReturn, MonthlyReturn, QuarterlyReturn, SemiAnnualReturn, YearlyReturn, MtdReturn, YtdReturn, YearlyRating, OneYearRating, AssetUnderManagement, TotalUnit, AumLastUpdate, LastFundFactSheet, LastUpdatedPortfolio, LastUpdate
sort_direction Type: string | Default: asc
Sort direction: asc or desc.
type Type: string
Filter mutual funds by the given type.
Possible values: balanced, equity, fixed_income, money_market, protected, limited_participation, reits, global
currency Type: string
Filter mutual funds by the given currency.
Possible values: idr or usd
categories Type: string
Filter mutual funds by the given categories.
Comma separated value from the following options: sharia, etf, index, conventional
fund_manager_id Type: int
Filter mutual funds by the given fund manager id. Get all possible fund manager ids from the List of Fund Manager.
custodian_bank_id Type: int:
Filter mutual funds by the given custodian bank id. Get all possible custodian bank ids from the List of Custodian Bank.
